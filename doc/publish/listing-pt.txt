O Mupen64Plus, Android Edition (AE), é um emulador de N64 para Android. No momento, está em beta testing, o que significa que é um trabalho ainda em constante progresso. Ainda há muitos problemas gráficos, de lag e áudio, mesmo nos dispositivos mais modernos. Visite www.paulscode.com para contribuir com os testes e desenvolvimento, assim como correção de erros e para ter respondidas as suas perguntas mais frequentes.

Para evitar problemas de Copyryght com as imagens de jogos, eu decidi colocar apenas jogos de dominio público e demos nas imagens. No entanto, se o emulador for removido da Play Store novamente, você ainda pode adquirir futuras atualizações de graça no site paulscode.com.

Este aplicativo está licensiado sob a GNU GPL v3, e o código completo está disponivel no repositório GIT público em https://github.com/paulscode/mupen64plus-ae, se algum desenvolvedor quiser ter acesso a escrever no repositório, entre em contato comigo (Paul) no fórum. Se você tiver problemas fazendo modificações no projeto, fale comigo, e eu posso te ajudar.

OTIMIZADO PARA O XPERIA PLAY
- Suporte de controle analógico pelo touchpad!

PERMISSÕES
Internet - A internet é usada para enviar relatório de erros. Ele NÃO baixa virus, spywares, ou propagandas de qualquer tipo. Taxas normais de dados aplicadas.
Gravar no armazenamento externo - O armazenamento do dispositivo é usado para salvar jogos e perfis de controles custumizados.
Vibrar - Se o seu dispositivo suportar, a vibração é usada para emular o Rumble Pak e feedback tátil dos botões.

Como jogar:
- Transfira as ROMS para qualquer pasta da memória interna ou cartão SD
- Encontre-as de dentro do emulador
- A pasta padrão para acesso ao cartão SD na maioria dos dispositivos é /mnt/sdcard
- As ROMS devem ter extensão .n64, .v64 ou .z64. Há, ainda, a alternativa de um arquivo .zip .
